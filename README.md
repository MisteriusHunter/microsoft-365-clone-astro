# 💻 Microsoft 365 Landing Page Clone (Pixel Perfect)

This project is a high-fidelity reconstruction of the Microsoft 365 landing page, focused on **extreme performance**, accessibility, and visual accuracy. Built using the **Astro** island architecture and **Tailwind CSS**.

## 🚀 Why this project?
- **Sub-second Loading**: Leveraging Astro's zero-JS by default approach to ensure lightning-fast speeds.
- **Pixel Perfect Execution**: Every margin, font-size, and color transition was meticulously crafted to match the original design.
- **Component-Driven Architecture**: Fully modular components using `.astro` files for high reusability.
- **Modern CSS**: Advanced use of Tailwind CSS for responsive layouts, custom gradients, and smooth UI interactions.

## 🛠️ Tech Stack
- **Framework**: Astro (Static Site Generator)
- **Styling**: Tailwind CSS (JIT Engine)
- **Icons & Assets**: Scalable SVG implementation and optimized image handling.
- **Fonts**: Microsoft's signature typography implementation.

## 📦 Key Components
- `PricingSection`: Dynamic pricing tables with custom highlights.
- `AppsGridSection`: Interactive grid showcasing Microsoft's app ecosystem.
- `FaqAccordion`: Accessible FAQ system using pure Astro logic.
